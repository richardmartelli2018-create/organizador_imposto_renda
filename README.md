# Organizador de Imposto de Renda em Excel

## Sobre o Projeto

Este projeto consiste na criação de uma ferramenta em Excel para auxiliar na organização e consolidação de dados necessários para a declaração de imposto de renda.

A planilha foi desenvolvida com foco em:

- Organização estruturada das informações
- Validações automáticas
- Menus de navegação
- Facilidade de uso
- Interface amigável

## Objetivo

Permitir que o usuário registre e acompanhe suas informações financeiras de forma organizada, reduzindo erros e facilitando o processo de declaração.

## Funcionalidades

- Cadastro de rendimentos
- Controle de despesas dedutíveis
- Consolidação automática de dados
- Validações para evitar erros de preenchimento
- Navegação facilitada entre abas

## Arquivo do Projeto

- `Organizador de IR.xlsx`

## Tecnologias Utilizadas

- Microsoft Excel
- Fórmulas e validações de dados
- Estruturação com múltiplas abas
